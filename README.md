# React Boilerplate

Custom React-Redux Boilerplate

## Dependencies Integrated

1. [node-sass](https://github.com/sass/node-sass)
2. [react-router-dom](https://reacttraining.com/react-router/web/guides/philosophy)
3. [axios](https://github.com/axios/axios)
4. [redux](https://redux.js.org/)
5. [react-redux](https://github.com/reduxjs/react-redux/tree/master/docs)
6. [redux-sagas](https://redux-saga.js.org/docs/introduction/BeginnerTutorial.html)
7. [redux-persist](https://github.com/rt2zz/redux-persist)
